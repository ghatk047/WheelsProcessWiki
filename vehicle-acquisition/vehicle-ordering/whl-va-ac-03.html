WHL-VA-AC-03

Emergency and Open-Market Vehicle Procurement

🏢 Vehicle Acquisition & Upfitting ➡ Vehicle Acquisition & Ordering 📝 6 L4 steps ✅ Complete
Description
This process at Wheels involves the emergency and open-market vehicle procurement for fleet management clients. It ensures that vehicles are quickly ordered and integrated into the fleet with minimal disruption.
Trigger
Immediate need for additional vehicles to replace damaged or retired units, or to expand the fleet due to increased demand.
Outcome
Successful acquisition and integration of new vehicles into the fleet, ensuring minimal downtime and compliance with regulatory requirements.

🎯 KPIs & Performance Targets
  • Vehicle uptime > 95%
  • Invoice accuracy > 99%
  • < 5 false positive fraud alerts per month
  • Setup completion within 24 hours of vehicle delivery
  • Title and registration completion within 3 days post-delivery
  • < 2 days for FNOL intake and initial assessment
⚠️ Operational Risks
  • Supply chain disruptions causing delays in vehicle availability.
  • Inaccurate order specifications leading to non-compliance with client needs.
  • Technical issues with integration platforms affecting order processing speed.
  • High volume of transactions overwhelming fraud detection systems.

Step Step Name (System) Role System System Action Input Output KPI Decision Exception Pain Point
1.1 FleetView VIM - Order Created Fleet Consultant FleetView API call to create a vehicle order in the FleetView Vehicle Inventory Management app. Order request with vehicle specifications. Vehicle order record in FleetView VIM. Order creation time < 2 hours N Y Manual data entry errors leading to delays.
1.2 SAP S/4HANA - Order Acknowledged Account Manager SAP S/4HANA ERP system acknowledges the order and updates the procurement module. Vehicle order record from FleetView VIM. Acknowledged order status in SAP S/4HANA. Order acknowledgment within 1 hour of creation. N Y SAP integration issues causing delays.
2.1 WEX Fuel Card API - Fraud Alert Fired Fuel Programme Manager WEX Fuel Card API Monitor and fire alerts for any fraudulent transactions. Transaction data from WEX Fuel Card. Fraud alert report to Account Manager. < 5 false positive alerts per month. Y N High volume of transactions leading to missed fraud detection.
2.2 ChargePoint - Public Charging Network Setup EV Consultant ChargePoint Set up public charging access for the new vehicle. Vehicle registration and driver information. Public charging network access configured. Setup completion within 24 hours of vehicle delivery. N Y Network connectivity issues causing delays.
3.1 Wheels Registration Services - Title & Registration Registration Specialist Wheels Registration Services Handle 50-state title and registration processes. Vehicle details from FleetView VIM. Registered vehicle with all necessary documentation. Title and registration completion within 3 days post-delivery. N Y Complex state regulations causing delays.

Incident reports from drivers. FNOL record in Wheels Accident Management Platform. < 2 days for FNOL intake and initial assessment. Y N High volume of accidents causing bottlenecks.
